# coding-teacher

A web app that teaches non-coders to read and understand code, on the premise that as AI writes more of our code, the ability to supervise becomes the most valuable skill.

## Goals

- Teach reading-comprehension of code, not just writing
- Use real, working examples rather than abstract exercises
- Give the learner immediate feedback that they have understood, not just memorised
- Start with HTML, CSS, and JavaScript because they run in any browser with no setup
